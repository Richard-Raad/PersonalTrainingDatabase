Report 1: Client Progress Summary

This report summarizes each client's workout history. It displays the total number of sessions logged, the total number of distinct exercises performed across all sessions, and the client's average Rate of Perceived Exertion (RPE) — a 1–10 scale indicating effort level. Clients with no sessions logged will show zeros, making this report useful for identifying both active and inactive users at a glance.

Report 2: Trainer Workload Summary

This report summarizes the workload and reach of each trainer in the system. It displays how many active clients each trainer is currently managing, the total number of workout plans they have assigned, and how many gyms they are affiliated with. This information is useful for administrators to identify overloaded trainers or trainers with capacity to take on new clients.
